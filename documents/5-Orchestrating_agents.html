A visual breakdown of how our AI agents store, process, and share information to perform complex tasks.

1. The Memory Block: The Agent's "Brain"

The foundation of an agent's memory is a simple Python dictionary.

This is where all core information is stored, like a database for the agent. It holds the agent's identity, instructions, and any shared knowledge it needs to perform its tasks consistently.

shared_memory = {
    "company": "The company is called AgentOS...",
    "persona": "You are responsible for evaluating candidates..."
}

2. The Formatting Process: The "Cheat Sheet"

The dictionary is converted into a structured text block for the AI.

The `get_formatted_memory()` function creates a "cheat sheet" that clearly separates different pieces of memory, making it easy for the AI model to understand its context and instructions for the current turn.

# Input (Dictionary)
{"company": "AgentOS"}
# Output (Formatted Text)
--- CORE MEMORY ---
<company>
"AgentOS"
</company>
‚Üì

3. The Full Prompt: The "Mission Briefing"

The formatted memory is combined with the persona and user request.

This is the final package of information sent to the Gemini model. It gives the agent its identity, its knowledge, and its task in a single, coherent prompt, ensuring the response is relevant and consistent.

// 1. System Prompt (Identity)
You are responsible for evaluating candidates...

// 2. Formatted Memory (Knowledge)
--- CORE MEMORY ---
<company>
"AgentOS"
</company>

// 3. User Message (Task)
**New Task:**
Evaluate this resume: ...

4. Shared Memory: The "Shared Whiteboard" ü§ù

Multiple agents can access the exact same memory block.

By passing the same `memory_blocks` dictionary to multiple agents, they share a common source of truth. This enables collaboration and consistency, ensuring all agents work with the most up-to-date information.

// Both agents reference the same dictionary in memory
eval_agent.memory_blocks = shared_memory
outreach_agent.memory_blocks = shared_memory

// Updating the dictionary affects both agents instantly
shared_memory['company'] = "Letta"
